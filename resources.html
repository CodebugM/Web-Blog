Other resources

Atom text editor

The Atom text editor is a great tool to edit all your code. With smart autocompletion and a bunch of packages you can install to customise Atom and make it work for you, coding is easy and quick.

Download and/or find out more about Atom on their website

GitHub

GitHub is the website to control the versions of your code, share your code and collaborate with others. Check out their website here and, if you like, check out my GitHub profile here.

"Learn Enough" authors Hartl and Donahoe regard git one of the three essential skills all developers need, along with Command Line and Text Editor. Find their Learn Enough Git to Be Dangerous tutorial here to learn all about version control, automatic ways to track changes in software projects, and view previous versions of projects.

Heroku

Mozilla Developer Network web docs

The good people over at MDN write "resources for developers, by developers". They are a very good website to refer to when looking for solutions to problems you may have with web technologies. A good tip is also to google a problem and put "mdn" into your search request to find the answer you are looking for in the MDN docs. Link to the MDN web docs

Stack Overflow

Codewars

Codewars is an online platform that teaches you a number of programming languages using real-life code challenges like C, C++, C#, JavaScript, Python, SQL, Swift, Ruby, the Shell etc. Join the Jedi here.

Command line

Whether you are using the command line, terminal on a Mac or Hyper, the command line is a very powerful tool for developers.

  • Hyper Terminal
  • Learn Enough Command Line to Be Dangerous

    • Learn enough command line to be dangerous is an introduction to the command line for complete beginners. It was written by Michael Hartl, the creator of the Ruby on Rails tutorial, one of the leading introductions to web development and cofounder and principal at LearnEnough.com, a cool platform teaching you "technical sophistication, the literacy of the 21st century" in a number of free online tutorials . Check out the full book on "Learn Enough Command Line to Be Dangerous" is available here.
